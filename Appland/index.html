A classification providing information on land cover, and the types of human activity involved in land use. It may also facilitate the assessment of environmental impacts on, and potential or alternative uses of land.

About

DeepSight is a free web service that provides land use and land cover semantics of satellite images using supervised learning.

About DeepSight

DeepSight proposes to apply different deep learning models to automatically classify/label images. It builds an efficient and more effective method for land use planning.

  • Learning Technique: Supervised Learning
  • Underlying Algorithm: Convolutional Neural Networks(CNN)
  • Access: Via API

The user sends a satellite image and DL model is applied to this image and the land use/land cover data is returned to the user.

About EuroSAT

EuroSAT is a dataset that is based on Sentinel-2 satellite images covering 13 spectral bands and consisting of 10 classes with 27000 labeled and geo-referenced samples.

  • Classes: 10
  • Number of Images: 2000-3000 per class
  • Patch Measure: 64x64 pixels.
  • Class Names: Industrial Buildings, Residential Buildings, Annual Crop, Permanent Crop, River, Sea & Lake, Herbaceous Vegetation, Highway, Pasture and Forest.
  • Number of Datasets: 2
  • Names of Datasets: RGB Band Combination, All Spectral Bands
  • Open and freely accessible dataset.
