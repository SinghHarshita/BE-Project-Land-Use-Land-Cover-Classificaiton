Get started

    API Endpoint

    https://deepsight.azurewebsites.net/predict/

The API takes image inputs from the user and returns the classified data back to the user. According to the request, a response will be generated and following that, its land cover will be stated.

At present, no API key is required to use this API. The API is free to use. There is no limit on the number of requests per day.

send image

                
                    
                

                
// Example JS Code to Make Request and Receive Response
APIDemoForm.onsubmit = async(e) => { 
    e.preventDefault();

    let body = new FormData()

    body.append('img', 
        document.getElementById('img').files[0], 
        document.getElementById('img').value
    )

    await fetch('https://deepsight.azurewebsites.net/predict/', {
        method: 'POST',
        // mode: 'no-cors',
        body: body
    }).then((response) => {
        if (response.status === 200) {
            response.json().then((res) => {
                console.log(res)
            }).catch((E) => {
                console.log(E)
            })
        }
    })
    .catch((e) => {
        console.log(e)
    })
}

To get classified data you need to make a POST call to the following url :
https://deepsight.azurewebsites.net/predict/


QUERY PARAMETERS

Field Type Description
img File Object The image file

Important Points

  • The image extension is tif or tiff
  • The image is of Sentinel-2 satellite
  • The image consists of all the 13 bands
  • The image dimensions are 64x64 pixels
                
Response JSON Example :

{
    class: "AnnualCrop"
}

Errors

The DeepSight API uses the following error codes:

Error Code Meaning
500 Internal Server Error. The image send doesn't have the right dimensions. The right dimensions are - 13bands, 64x64 pixels.
400 Bad Request.
